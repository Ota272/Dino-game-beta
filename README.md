# Dino game

Клон игры в духе «динозаврика» из Chrome: бег, прыжки, кактусы и птицы, счёт и рекорд. Реализовано на **Python** и **Pygame**.

## Требования

- Python 3.8+ (рекомендуется актуальная 3.x)
- [Pygame](https://www.pygame.org/)

## Установка

В папке проекта:

```bash
pip install pygame
```

## Файлы рядом с `main.py`

Игра ожидает в **той же папке**, что и `main.py`, следующие файлы:

| Файл | Назначение |
|------|------------|
| `dino.png` | Спрайт динозавра |
| `cactus.png` | Кактус (препятствие на земле) |
| `bird.png` | Птица (препятствие выше) |
| `ground.png` | Текстура земли (растягивается по ширине окна) |
| `PressStart2P-Regular.ttf` | Шрифт для текста счёта и меню |
| `jump_sound.wav` | Звук прыжка |
| `score_sound.wav` | Звук при начислении очка |

Если звуковые файлы отсутствуют или повреждены, игра всё равно запустится: прыжок и очки будут без звука (в консоли может появиться сообщение об ошибке загрузки).

## Запуск

```bash
python main.py
```

## Управление

- **Пробел** — начать игру / перезапуск после проигрыша
- **Пробел** во время игры — прыжок

## Геймплей

- Препятствия появляются по таймеру; это случайный кактус или птица (у птицы другая высота и скорость).
- Очко начисляется, когда препятствие оказывается слева от динозавра.
- Отображаются текущий **Score** и **Record** за сессию.
- При счёте **100 и выше** фон становится зелёным.
- Столкновение с препятствием — конец раунда; снова **Пробел** для новой попытки.

## Окно

Размер окна: **900×450** пикселей, заголовок окна: «Dino game», целевая частота кадров: **30 FPS**.
